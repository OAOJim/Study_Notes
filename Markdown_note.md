## Centering Text
To center text in Markdown:
```html
<center> text need to be centered </center>
```
Center the whole file:
```html
<dev align ="center"> text need to be centered </center>
```

## Mathematical Expressions
Use \$ Mathematical expressions in LaTex \$ to get mathematical expressions. 

Note: in GitHub use
```latex
```math
```
For the \begin{} and \end{} function 

### Matrix
```math          
\begin{matrix}
a & b & c \\
c & d & e \\
f & g & h \\
\end{matrix}
```
```latex
\begin{matrix}
a & b & c \\
c & d & e \\
f & g & h \\
\end{matrix}
```

```math 
\begin{bmatrix}
a & b & c \\
c & d & e \\
f & g & h \\
\end{bmatrix}
```

```latex
\begin{bmatrix}
a & b & c \\
c & d & e \\
f & g & h \\
\end{bmatrix}
```

### Cases
```math 
\begin{cases}
a = b + c \\
b = a + c \\
c = a + b \\
\end{cases}
```

```latex
\begin{cases}
a = b + c \\
b = a + c \\
c = a + b \\
\end{cases}
```
