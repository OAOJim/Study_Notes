# Markdown note


## Centering Text
To center text in Markdown, you can use HTML tags:
```html
<p align="center">This text is centered.</p>
```
or you can do 
```html
<center> text need to be centered </center>
```

## Mathematical Expressions
Use \$ Mathematical expressions in LaTex \$ to get mathematical expressions. 

Note: in GitHub use
```latex
```math
```
For the \begin{} and \end{} function 

### Matrix
```math 
\begin{bmatrix}
a & b & c \\
c & d & e \\
f & g & h \\
\end{bmatrix}
```

```latex
\begin{bmatrix}
a & b & c \\
c & d & e \\
f & g & h \\
\end{bmatrix}
```

### Cases
```math 
\begin{cases}
a = b + c \\
b = a + c \\
c = a + b \\
\end{cases}
```

```latex
\begin{cases}
a = b + c \\
b = a + c \\
c = a + b \\
\end{cases}
```
